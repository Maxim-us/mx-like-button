"MX Like Button" will add a mechanism of "like buttons" similar to Facebook to your Buddypress website.

== Description ==

Your users will be able to express emotions at a particular post in the stream of activity. This plugin is similar to the Facebook "Like system" but very simplified.

== Installation ==

= From your WordPress dashboard =

1. Visit 'Plugins > Add New'
2. Search for 'MX Like Button'
3. Activate the plugin from your Plugins page.

= From WordPress.org =

1. Download 'MX Like Button'.
2. Upload the 'MX Like Button' directory to your '/wp-content/plugins/' directory, using your favorite method (ftp, sftp, scp, etc...)
3. Activate 'MX Like Button' from your Plugins page.
